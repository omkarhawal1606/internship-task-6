## Portfolio Website (Flask)

## This project is a personal portfolio website built using Flask, a Python web framework. It displays your introduction, skills, projects, and contact information in a clean and simple layout.

## Features

1.Home page with basic introduction

2.Projects page to show your work

3.Resume page with your details

4.Contact page with a message form

5.Responsive design for mobile and desktop

## Technologies Used

i.Backend: Flask (Python)

ii.Frontend: HTML, CSS, Bootstrap

iii.Optional database: SQLite

## Setup

## Download or clone the project.
## Create a virtual environment, install the required libraries, and run the Flask application to view the website in your browser.

## Folder Structure

The project includes main files like app.py, a templates folder for HTML pages, and a static folder for CSS, images, and scripts.

## Future Improvements

## You can add features like a blog, animations, and email sending from the contact form.
